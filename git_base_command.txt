Git 撤销 commit
	git reset --hard commit_id
	彻底回退到某个版ben
修改最后一次提交
	git commit --amend 重新提交
修改已经暂存的文件
	git reset HEAD <file>
取消对文件的修改
	git checkout <file>
git log 
	-p 选项展开显示每次提交的内容差异
	-2 则仅现实最近的两侧更新.
远程仓库的使用
	察看当前的远程库
		git remote -v
	克隆
		git clone url
 	添加
		git remote add name url
	抓去仓库信息
		git fetch name
	push
		git push -u remote-name brach-name
	远程仓库的删除和重命名
		git remote name name1 name2
		git remote rm rem://github.com/leiding1001/backup.git remtoe-name
git标签
	显示已有的标签
		git tag
	新建标签
		含注释的标签
		git tag -a v.14 -m "dddd"	
	察看相应的版本信息	
		git show v.14
	签署标签
		
//删除远程分支
	git push --delete origin devel
//重命名远程分支
	git branch -m devel develop
//察看远程分支
	git branch -a
//删除tag
	git push origin --delete tag<name>
